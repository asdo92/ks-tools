ks-utfs - Patch UTF-8 subtitles file(s) replacing bad characters
================================================================

### Syntax:

```shell
$ ks-utfs </absolute/path/subtitle.srt>
```

### Example:

```shell
$  ks-utfs /data/movies/Spiderman.srt
```

_Note: This tool is automatically integrated into the `ks-mp4`, `ks-mp4s`, `ks-mp4k`, `ks-av1`, `ks-av1s` & `ks-avi` tools._
